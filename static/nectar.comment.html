>>> from nectar.account import Account
>>> from nectar import Steem

>>> postdate = c["created"]

get_curation_rewards(pending_payout_SBD=False, pending_payout_value=None)ÔÉÅ

Returns the curation rewards. The split between creator/curator is currently 50%/50%.

Parameters:

  • pending_payout_SBD (bool) ‚Äì If True, the rewards are returned in SBD and not in STEEM (default is False)

  • pending_payout_value (float, str) ‚Äì When not None this value instead of the current value is used for calculating the rewards

pending_rewards is True when the post is younger than 7 days. unclaimed_rewards is the amount of curation_rewards that goes to the author (self-vote or votes within the first 30 minutes). active_votes contains all voter with their curation reward.
